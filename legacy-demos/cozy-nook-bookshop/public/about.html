Our Story

A legacy bookstore designed for testing Modern Web Guidance.

The Legacy Playground

Welcome to the Cozy Nook! While it looks like a peaceful place to find books, this application is actually built to be slow and outdated. It is an app where developers can find common web performance mistakes and fix them using modern techniques.

Test Modern Web Guidance

Use Modern Web Guidance and Chrome DevTools for agents to audit this site. Run a Lighthouse performance audit on mobile, locate the loading bottlenecks, and practice upgrading this legacy code to modern, native browser features!

Identify and Fix the Bottlenecks

🐌

Slow Image Priority

The main book cover is deprioritized, while offscreen books load eagerly. Can you fix this LCP bottleneck?

⚙️

Heavy JavaScript Animations

The carousel uses custom JavaScript for motion. Upgrade it to native CSS Scroll-Driven Animations.

🚪

Legacy Menu Toggles

The sidebar drawer is ripe for modernization. Replace custom JavaScript toggles with the native Popover API.

🪄

Missing Page Transitions

Book details load instantly without any flare. Integrate the View Transitions API for smooth layout changes.
